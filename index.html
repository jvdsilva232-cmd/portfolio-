Sistemas construídos para operação real

Três ferramentas nascidas de problemas reais: validar → qualificar → distribuir. Todas rodam 100% localmente, sem enviar dados a serviços externos.

// projeto 01

GPON Processor

Bases enormes chegam com colunas fora de padrão e registros inválidos. O GPON Processor trata e valida bases para consulta de cobertura sem depender de internet.

  • Detecta colunas de CNPJ, CEP, número e endereço automaticamente
  • Normaliza, valida e descarta registros fora dos critérios
  • Processa em blocos e divide saídas acima de 200 mil linhas
pythonpandasweb localoffline
// projeto 02

Qualificador de Telefones

Listas de contatos misturam celulares, fixos, 0800 e lixo. O Qualificador limpa e classifica bases telefônicas brasileiras, prontas para campanhas e discadores.

  • Classifica móvel, fixo, sem DDD, especial e inválido com validação de DDDs
  • Três interfaces: painel web, desktop e terminal
  • Exportação adaptativa em Excel ou CSV + relatório estatístico
pythonpandasflasktkinterjs
// projeto 03

Radar de Cidades

Dezenas de planilhas heterogêneas viram bases limpas, separadas por cidade e distribuídas em lotes prontos para o time de vendas.

  • ETL completo: leitura recursiva, deduplicação e bloqueio de clientes já vendidos
  • Excel formatado por cidade com filtros e cabeçalho congelado
  • API HTTP e painel web construídos só com a biblioteca padrão do Python
pythonopenpyxlstdlib httpthreading
